# covid-vue-app


## Project setup

```

npm install

```


## Develop in a containerised environment

This serves a hot reload server serving Vue application in a image, accessible at `localhost:8080`

```

docker-compose up

```

Install `docker-compose` [here](https://docs.docker.com/compose/install/)
  

# [Not recommended]
